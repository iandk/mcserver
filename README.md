# mcserver
Simple bash script to control a Minecraft server


## Installation
```
wget -qO /usr/local/bin/mcserver https://git.io/JtYSi
chmod +x /usr/local/bin/mcserver
```



## Available commands
Start minecraft server
```
mcserver start
```
Attach console
```
mcserver attach
```
Stop server
```
mcserver stop
```
Show server status
```
mcserver status
```
Backup server
```
mcserver backup
```
